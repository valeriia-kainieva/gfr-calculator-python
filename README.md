# GFR Calculator (CKD-EPI)

This project implements a Glomerular Filtration Rate (GFR) calculator based on the CKD-EPI equation.

## Overview
The tool estimates kidney function using clinical parameters:
- Age
- Sex
- Serum creatinine

## Features
- GFR calculation using CKD-EPI formula
- Simple input interface
- Clinical interpretation of results

## Clinical Relevance
GFR is a key indicator of kidney function and is widely used in clinical practice to assess chronic kidney disease (CKD).

## Example
Input:
Age: 45  
Sex: Female  
Creatinine: 1.0 mg/dL  

Output:
Estimated GFR: XX mL/min/1.73m²
